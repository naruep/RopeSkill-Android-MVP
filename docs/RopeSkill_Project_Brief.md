# RopeSkill — Android MVP Project Brief

อัปเดตล่าสุด: 22 กรกฎาคม 2026

## ภาพรวม

RopeSkill คือแอป Android ที่ใช้กล้องและ AI Pose Estimation เพื่อช่วยผู้ใช้ฝึกกระโดดเชือก ตรวจจับท่าทาง และนับการกระโดด โดย MVP จะเริ่มจากท่า Basic Bounce และประมวลผลข้อมูลบนอุปกรณ์เป็นหลัก

## ปัญหาที่ต้องการแก้

ผู้ฝึกกระโดดเชือกด้วยตนเองอาจนับจำนวนครั้งได้ยาก ไม่ทราบจังหวะการฝึกที่สม่ำเสมอ และขาดข้อมูลที่ใช้ประเมินผลหลังจบการฝึก

## กลุ่มผู้ใช้เป้าหมาย

- ผู้เริ่มต้นฝึกกระโดดเชือก
- ผู้ที่ต้องการนับจำนวนครั้งและเวลาโดยไม่ใช้อุปกรณ์เสริม
- ผู้ทดสอบ MVP บนโทรศัพท์ Android จริง

## คุณค่าหลักของ MVP

- เปิดกล้องและเห็นภาพแบบเรียลไทม์
- แสดงจุด Pose Landmarks ที่ใช้งานได้
- ตรวจจับและนับ Basic Bounce
- จบ Session และดูผลการฝึก
- บันทึกผลไว้ในอุปกรณ์

## ขอบเขตปัจจุบัน

อยู่ในขอบเขต:

- Android Native ด้วย Kotlin
- Jetpack Compose และ Material 3
- CameraX
- MediaPipe Pose Landmarker
- Local storage
- การทดสอบบนโทรศัพท์ Android จริง

ยังไม่อยู่ในขอบเขต:

- Backend และบัญชีผู้ใช้
- Cloud processing หรือการอัปโหลดภาพกล้อง
- iOS
- Payment, cryptocurrency หรือ blockchain
- ระบบแข่งขันและคะแนนแบบสมบูรณ์

## ตัวชี้วัดความสำเร็จ

- แอปทำงานได้อย่างเสถียรบนโทรศัพท์ Android จริง โดยต้องรอผู้ใช้ยืนยัน
- Camera preview ต่อเนื่องและไม่ค้าง
- Pose landmarks ใช้งานได้ในสภาพทดสอบที่กำหนด
- Basic Bounce ถูกนับด้วยความแม่นยำที่วัดและบันทึกได้
- Session ถูกบันทึกและเรียกดูได้
- ไม่มีการเก็บหรือส่งภาพกล้องออกนอกอุปกรณ์โดยไม่ได้รับอนุมัติ

## ข้อสมมติฐาน

- เริ่มพัฒนา Android ก่อน iOS
- ใช้กล้องหลังเป็นค่าเริ่มต้นในช่วงทดลอง เว้นแต่การทดสอบพบว่ากล้องหน้าเหมาะกว่า
- การประมวลผลกล้องและ Pose จะเกิดบนอุปกรณ์
- เกณฑ์ความแม่นยำขั้นต่ำจะกำหนดหลังมีข้อมูลทดสอบจริง

## สถานะล่าสุด

- ยืนยันแนวทาง Kotlin Native สำหรับ Android MVP แล้ว
- ยืนยันลำดับการพัฒนาแบบทีละขั้นแล้ว
- ยังไม่ได้ตรวจสอบโครงสร้าง source code หรือผลการ Build ล่าสุด
- ยังไม่มีผลทดสอบบนโทรศัพท์จริงที่ได้รับการยืนยัน
